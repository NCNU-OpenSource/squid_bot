## 事前準備
1. 在 Telegram 上加 `@BotFather` 為好友
2. `/newbot` 設定 `name` `username`
3. 會得到 Bot 的 `token`，請自行保留好，請勿外流！![](https://i.imgur.com/Hu4k9ha.jpg)
4. `/setcommands` 設定你的 bot 要有哪些功能，也可以之後再加

## 環境準備
- python3
- `model`：存放訓練好的 model
- `main.py`：bot 主程式
- `Identify.py`：辨識圖片
